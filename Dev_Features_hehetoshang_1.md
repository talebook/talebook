# 可执行开发任务清单

## 一、UI统一与注册页改造 (已完成)
- **前端任务**：为 `login.vue` 与 `register.vue` 页面添加例如 `index.vue` 同款的顶部栏、侧边栏组件
- **逻辑适配**：将 `index.vue` 中顶部栏、侧边栏的交互逻辑迁移至 `login.vue` 与 `register.vue` 
- **验证**：确保三个页面的导航、样式和交互体验完全统一

## 二、私人图书馆密码界面优化
### 前端部分
- 在密码输入框的错误提示上方，新增文字提示显示区域
- 实现逻辑：当密码输入错误次数达到后台配置的 `x` 值时，自动显示该文字提示
- 实现逻辑：当密码输入正确时，自动隐藏该文字提示
- 样式适配：确保文字提示的样式与整体页面风格统一

### Admin 后台配置
- 在 Admin 的「私人图书馆设置」页面，添加“文字提示内容”输入框
- 新增“错误次数阈值 `x`”输入框，用于配置触发显示的错误次数
- 保存配置到后端，供前端获取并使用

---

## 三、OPDS 设置功能开发
- **前端任务**：在设置页面新增「OPDS 设置」模块
- **交互实现**：添加 OPDS 开关控件，控制功能的开启/关闭状态
- **逻辑适配**：
  - 关闭状态：访问 `/opds` 接口返回 404，且隐藏 OPDS 帮助入口
  - 开启状态：保持 OPDS 接口和帮助入口的正常功能
- **后端配合**：接口需根据开关状态返回对应响应
